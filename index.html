HTML

(Voltar)

Html é uma linguagem de marcação e texto para desenvolver sites.

Ela é a base de todo site, todo dev que queira desenvolver em Front tem que aprender HTML

HEAD

(Voltar)

É o local onde se declara as informações do site, linguagem, links etc.

BODY

(Voltar)

O Body é onde o site é desenvolvido onde o usuário vai obter as informações do site.

TAGS

(Voltar)

São elementos utilizados para informar ao navegador que tipo de estrutura é essa que está sendo construída, podendo ser títulos, parágrafos, imagens, links, entre outros.

FOOTER

(Voltar)

É o rodapé do site fora do Body, geralmente é usado para contatos no site
